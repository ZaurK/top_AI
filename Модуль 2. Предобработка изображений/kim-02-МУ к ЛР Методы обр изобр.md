### 2. Теоретические сведения

#### 2.1. Фильтрация изображений
- **Линейные фильтры** – усредняющий, гауссов – удаляют шум, но размывают границы.  
- **Нелинейные фильтры** – медианный – эффективно подавляют импульсный шум, сохраняя резкость краёв.  
- **Повышение резкости** – выделение высокочастотных составляющих (оператор Лапласа, маска нерезкости) и их добавление к изображению.

#### 2.2. Геометрические преобразования
- **Поворот** – вращение вокруг центра на заданный угол (`cv2.getRotationMatrix2D` + `cv2.warpAffine`).  
- **Масштабирование** – изменение размеров с заданным интерполяционным методом (`cv2.resize`).  
- **Аффинные преобразования** – сдвиг, поворот, масштабирование, отражение (`cv2.warpAffine`).  
- **Перспективные преобразования** – коррекция искажений, изменение угла обзора (`cv2.warpPerspective`).  
- Применяются для нормализации изображений, устранения геометрических искажений, аугментации данных.

#### 2.3. Пороговая обработка
- **Бинаризация** – преобразование изображения в чёрно-белое по фиксированному порогу (`cv2.threshold`).  
- **Адаптивная пороговая обработка** – вычисление порога для каждого пикселя на основе локального окружения (`cv2.adaptiveThreshold`). Полезна при неравномерном освещении.  
- Применяется для сегментации объектов, выделения контуров, упрощения изображения для дальнейшего анализа.

#### 2.4. Улучшение контраста
- **Эквализация гистограммы** – выравнивание яркостного распределения для улучшения контраста. Для цветных изображений применяется к каналу яркости (V в HSV или L в LAB).  
- **CLAHE** – адаптивная версия, ограничивающая усиление шума.

#### 2.5. Цветовые пространства
- **RGB** – базовое пространство, каналы коррелированы.  
- **HSV** – разделяет оттенок, насыщенность и яркость.  
- **LAB** – восприятие близко к человеческому; канал L – яркость.

#### 3.3. Применение методов обработки

**Каждый метод применяется к копии исходного изображения.**

##### А. Фильтрация
1. **Гауссово размытие** – `cv2.GaussianBlur` с разными размерами ядра (3×3, 5×5, 7×7).
2. **Медианная фильтрация** – `cv2.medianBlur` с разными размерами окна.
3. **Повышение резкости** – маска нерезкости: `sharp = orig + α*(orig - blurred)`.

##### Б. Геометрические преобразования
1. **Поворот** – на 30°, 45°, 90° (с изменением размера или без).
2. **Масштабирование** – увеличение и уменьшение в 0.5, 1.5, 2 раза с различной интерполяцией.
3. **Аффинное преобразование** – сдвиг, отражение по горизонтали/вертикали.
4. **Перспективное преобразование** – задать четыре точки и преобразовать в прямоугольник (для коррекции искажений).

##### В. Пороговая обработка
1. **Глобальная бинаризация** – `cv2.threshold` с разными порогами (Otsu – автоматический подбор порога).
2. **Адаптивная пороговая обработка** – `cv2.adaptiveThreshold` с разными размерами блока и константами C.
3. Применить к полутоновому изображению (преобразовать в оттенки серого).

##### Г. Улучшение контраста
1. **Эквализация гистограммы** – через HSV (канал V) или LAB (канал L).
2. **CLAHE** – `cv2.createCLAHE` с разными параметрами clipLimit и tileGridSize.

#### 3.4. Оценка качества
- Для фильтрации и улучшения контраста – вычислить **MSE**, **PSNR**, **SSIM** относительно исходного (если оно без шума).
- Для геометрических преобразований – оценить визуально, а также можно сравнить с эталоном (если есть).
- Для пороговой обработки – визуальная оценка качества сегментации.

### 5. Примерный код (шаблон)

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt
from skimage.metrics import structural_similarity as ssim

# Загрузка и отображение
img = cv2.imread('image.jpg')
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

def show(img, title):
    plt.figure(figsize=(6,6))
    plt.imshow(img)
    plt.title(title)
    plt.axis('off')
    plt.show()

show(img_rgb, 'Original')

# ----- Фильтрация -----
blur = cv2.GaussianBlur(img_rgb, (5,5), 0)
median = cv2.medianBlur(img_rgb, 5)
kernel = np.array([[-1,-1,-1],[-1,9,-1],[-1,-1,-1]])
sharp = cv2.filter2D(img_rgb, -1, kernel)

# ----- Геометрические преобразования -----
(h, w) = img_rgb.shape[:2]
center = (w//2, h//2)
M = cv2.getRotationMatrix2D(center, 45, 1.0)
rotated = cv2.warpAffine(img_rgb, M, (w, h))

resized = cv2.resize(img_rgb, None, fx=1.5, fy=1.5, interpolation=cv2.INTER_LINEAR)
flipped = cv2.flip(img_rgb, 1)  # горизонтальное отражение

# ----- Пороговая обработка (на оттенках серого) -----
gray = cv2.cvtColor(img_rgb, cv2.COLOR_RGB2GRAY)
_, thresh_otsu = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
thresh_adapt = cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                     cv2.THRESH_BINARY, 11, 2)

# ----- Улучшение контраста -----
def equalize_hsv(img):
    hsv = cv2.cvtColor(img, cv2.COLOR_RGB2HSV)
    hsv[:,:,2] = cv2.equalizeHist(hsv[:,:,2])
    return cv2.cvtColor(hsv, cv2.COLOR_HSV2RGB)

eq = equalize_hsv(img_rgb)

def clahe_lab(img, clip=2.0, grid=(8,8)):
    lab = cv2.cvtColor(img, cv2.COLOR_RGB2LAB)
    l, a, b = cv2.split(lab)
    clahe = cv2.createCLAHE(clipLimit=clip, tileGridSize=grid)
    l2 = clahe.apply(l)
    lab2 = cv2.merge((l2, a, b))
    return cv2.cvtColor(lab2, cv2.COLOR_LAB2RGB)

clahe_img = clahe_lab(img_rgb)

# Визуализация результатов
show(blur, 'Gaussian blur')
show(median, 'Median filter')
show(sharp, 'Sharpened')
show(rotated, 'Rotated 45°')
show(resized, 'Resized 1.5x')
show(flipped, 'Flipped')
show(thresh_otsu, 'Otsu threshold')
show(thresh_adapt, 'Adaptive threshold')
show(eq, 'Histogram equalization')
show(clahe_img, 'CLAHE')

# Метрики для фильтрации и улучшения (если есть эталон)
print('PSNR blur:', cv2.PSNR(img_rgb, blur))
print('SSIM blur:', ssim(img_rgb, blur, multichannel=True, data_range=255))
# аналогично для других методов
```
