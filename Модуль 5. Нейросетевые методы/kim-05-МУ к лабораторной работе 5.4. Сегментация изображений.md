### 2. Теоретические сведения

#### 2.1. Задачи сегментации

**Семантическая сегментация** — классификация каждого пикселя изображения по принадлежности к определённому классу.

**Инстансная сегментация** — отдельное выделение каждого объекта на изображении.

**Паноптическая сегментация** — объединение семантической и инстансной сегментации.

#### 2.2. Архитектуры для сегментации

**U-Net**:
- Состоит из **сжимающего пути** (encoder) и **расширяющего пути** (decoder);
- **Skip-connections** — передача карт признаков с encoder на decoder;
- Позволяет восстанавливать пространственную информацию.

**Структура U-Net:**
```
Encoder (сжатие):           Decoder (расширение):
Input -> Conv -> Pool ->    Up-conv -> Conv -> Skip-connection
         Conv -> Pool ->    Up-conv -> Conv -> Skip-connection
         Conv -> Pool ->    Up-conv -> Conv -> Skip-connection
         Conv -> Pool ->    Up-conv -> Conv -> Skip-connection
         Bottleneck         Output (mask)
```

**FCN (Fully Convolutional Network)**:
- Заменяет полносвязные слои на свёрточные;
- Использует транспонированную свёртку для восстановления размера;
- Применяет skip-connections для улучшения детализации.

**DeepLab**:
- Использует **Atrous Convolution** (дырчатая свёртка) для увеличения receptive field без потери разрешения;
- Применяет **ASPP (Atrous Spatial Pyramid Pooling)** для захвата контекста на разных масштабах.

#### 2.3. Метрики качества сегментации

**IoU (Intersection over Union)**:
$$
IoU = \frac{TP}{TP + FP + FN}
$$

**mIoU (mean IoU)**:
$$
mIoU = \frac{1}{N} \sum_{i=1}^{N} IoU_i
$$

**Pixel Accuracy**:
$$
PA = \frac{TP + TN}{TP + TN + FP + FN}
$$

   ```python
           images.append(img)
           
           # Загрузка маски
           mask_file = img_file.replace('.jpg', '_mask.png')
           mask_path = os.path.join(mask_dir, mask_file)
           mask = cv2.imread(mask_path, cv2.IMREAD_GRAYSCALE)
           mask = cv2.resize(mask, image_size, interpolation=cv2.INTER_NEAREST)
           masks.append(mask)
       
       return np.array(images), np.array(masks)
   ```

#### 3.2. Реализация модели U-Net

```python
def unet_model(input_size=(128, 128, 3), num_classes=3):
    inputs = tf.keras.layers.Input(input_size)
    
    # Encoder (сжатие)
    # Layer 1
    conv1 = tf.keras.layers.Conv2D(64, 3, activation='relu', padding='same')(inputs)
    conv1 = tf.keras.layers.Conv2D(64, 3, activation='relu', padding='same')(conv1)
    pool1 = tf.keras.layers.MaxPooling2D(pool_size=(2, 2))(conv1)
    
    # Layer 2
    conv2 = tf.keras.layers.Conv2D(128, 3, activation='relu', padding='same')(pool1)
    conv2 = tf.keras.layers.Conv2D(128, 3, activation='relu', padding='same')(conv2)
    pool2 = tf.keras.layers.MaxPooling2D(pool_size=(2, 2))(conv2)
    
    # Layer 3
    conv3 = tf.keras.layers.Conv2D(256, 3, activation='relu', padding='same')(pool2)
    conv3 = tf.keras.layers.Conv2D(256, 3, activation='relu', padding='same')(conv3)
    pool3 = tf.keras.layers.MaxPooling2D(pool_size=(2, 2))(conv3)
    
    # Layer 4
    conv4 = tf.keras.layers.Conv2D(512, 3, activation='relu', padding='same')(pool3)
    conv4 = tf.keras.layers.Conv2D(512, 3, activation='relu', padding='same')(conv4)
    pool4 = tf.keras.layers.MaxPooling2D(pool_size=(2, 2))(conv4)
    
    # Bottleneck
    conv5 = tf.keras.layers.Conv2D(1024, 3, activation='relu', padding='same')(pool4)
    conv5 = tf.keras.layers.Conv2D(1024, 3, activation='relu', padding='same')(conv5)
    
    # Decoder (расширение)
    # Up Layer 1
    up6 = tf.keras.layers.Conv2DTranspose(512, 2, strides=(2, 2), padding='same')(conv5)
    merge6 = tf.keras.layers.concatenate([conv4, up6])
    conv6 = tf.keras.layers.Conv2D(512, 3, activation='relu', padding='same')(merge6)
    conv6 = tf.keras.layers.Conv2D(512, 3, activation='relu', padding='same')(conv6)
    
    # Up Layer 2
    up7 = tf.keras.layers.Conv2DTranspose(256, 2, strides=(2, 2), padding='same')(conv6)
    merge7 = tf.keras.layers.concatenate([conv3, up7])
    conv7 = tf.keras.layers.Conv2D(256, 3, activation='relu', padding='same')(merge7)
    conv7 = tf.keras.layers.Conv2D(256, 3, activation='relu', padding='same')(conv7)
    
    # Up Layer 3
    up8 = tf.keras.layers.Conv2DTranspose(128, 2, strides=(2, 2), padding='same')(conv7)
    merge8 = tf.keras.layers.concatenate([conv2, up8])
    conv8 = tf.keras.layers.Conv2D(128, 3, activation='relu', padding='same')(merge8)
    conv8 = tf.keras.layers.Conv2D(128, 3, activation='relu', padding='same')(conv8)
    
    # Up Layer 4
    up9 = tf.keras.layers.Conv2DTranspose(64, 2, strides=(2, 2), padding='same')(conv8)
    merge9 = tf.keras.layers.concatenate([conv1, up9])
    conv9 = tf.keras.layers.Conv2D(64, 3, activation='relu', padding='same')(merge9)
    conv9 = tf.keras.layers.Conv2D(64, 3, activation='relu', padding='same')(conv9)
    
    # Output
    output = tf.keras.layers.Conv2D(num_classes, 1, activation='softmax')(conv9)
    
    model = tf.keras.Model(inputs=inputs, outputs=output)
    return model
```

#### 3.5. Вычисление метрик

```python
def compute_iou(y_true, y_pred, num_classes):
    ious = []
    for cls in range(num_classes):
        y_true_cls = (y_true == cls).astype(int)
        y_pred_cls = (y_pred == cls).astype(int)
        intersection = np.sum(y_true_cls * y_pred_cls)
        union = np.sum(y_true_cls) + np.sum(y_pred_cls) - intersection
        iou = intersection / (union + 1e-8)
        ious.append(iou)
    return ious

def compute_metrics(y_true, y_pred, num_classes):
    ious = compute_iou(y_true, y_pred, num_classes)
    miou = np.mean(ious)
    return ious, miou

# Пример вычисления метрик
y_true = np.argmax(y_val_cat, axis=-1)
y_pred = model.predict(X_val)
y_pred = np.argmax(y_pred, axis=-1)

print(f"IOUs: {ious}")
print(f"mIoU: {miou:.4f}")
```
