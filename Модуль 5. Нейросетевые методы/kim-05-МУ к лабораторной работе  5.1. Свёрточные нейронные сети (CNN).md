### 2. Теоретические сведения

#### 2.1. Архитектура свёрточной нейронной сети

Свёрточные нейронные сети представляют собой класс глубоких нейронных сетей, специально разработанных для обработки данных с сетчатой структурой (изображения, временные ряды).

**Основные компоненты CNN:**

1. **Свёрточный слой (Conv2D)**: выполняет операцию свёртки входного изображения с набором обучаемых ядер (фильтров). Формула свёртки для двумерного случая:
   $$
   (I * K)(i,j) = \sum_{m=0}^{M-1}\sum_{n=0}^{N-1} I(i+m, j+n) \cdot K(m,n)
   $$
   где \(I\) — входное изображение, \(K\) — ядро свёртки размером \(M \times N\).

2. **Функции активации**:
   - **ReLU**: $f(x) = \max(0, x)$ — обеспечивает нелинейность и решает проблему затухающих градиентов.
   - **Sigmoid**: $f(x) = \frac{1}{1 + e^{-x}}$ — используется в выходных слоях для бинарной классификации.
   - **Softmax**: $f(x_i) = \frac{e^{x_i}}{\sum_{j} e^{x_j}}$ — для многоклассовой классификации.

3. **Подвыборка (Pooling)**:
   - **Max Pooling**: выбирает максимальное значение в окне.
   - **Average Pooling**: вычисляет среднее значение в окне.
   $$
   y_{i,j} = \max_{p=0}^{P-1}\max_{q=0}^{Q-1} x_{i+p, j+q}
   $$

4. **Полносвязные слои (Dense)**: все нейроны связаны со всеми нейронами предыдущего слоя.

5. **Регуляризация**:
   - **Dropout**: случайное "выключение" нейронов во время обучения с вероятностью \(p\).
   - **Batch Normalization**: нормализация активаций внутри батча.

#### 2.2. Процесс обучения CNN

Обучение CNN осуществляется с помощью алгоритма обратного распространения ошибки (backpropagation) и стохастического градиентного спуска (SGD) или его модификаций (Adam, RMSprop).

**Функция потерь** для многоклассовой классификации:
$$
L = -\sum_{i=1}^{N} y_i \log(\hat{y}_i)
$$
где \(y_i\) — истинная метка, $\hat{y}_i$ — предсказанная вероятность.

**Метрики качества**:
- **Accuracy**: доля правильных ответов.
- **Precision**: $ \frac{TP}{TP + FP} $
- **Recall**: $ \frac{TP}{TP + FN} $
- **F1-score**: $ 2 \cdot \frac{Precision \cdot Recall}{Precision + Recall} $

#### 2.3. Аугментация данных

Аугментация — техника увеличения размера обучающей выборки путём применения случайных преобразований к изображениям:
- Повороты (на случайный угол);
- Сдвиги (по горизонтали и вертикали);
- Масштабирование;
- Отражения (горизонтальные/вертикальные);
- Изменение яркости и контрастности;
- Добавление шума.

model = models.Sequential([

    layers.BatchNormalization(),

    layers.Conv2D(64, (3, 3), activation='relu'),

    layers.Dense(10, activation='softmax')

#### 3.7. Тестирование на реальных изображениях

```python
import cv2

def preprocess_image(image_path, target_size=(32, 32)):
    img = cv2.imread(image_path)
    img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
    img = cv2.resize(img, target_size)
    img = img.astype('float32') / 255.0
    img = np.expand_dims(img, axis=0)
    return img

def predict_image(model, image_path, class_names):
    img = preprocess_image(image_path)
    prediction = model.predict(img)
    predicted_class = np.argmax(prediction)
    confidence = np.max(prediction)
    return class_names[predicted_class], confidence

# Пример использования
class_names = ['airplane', 'automobile', 'bird', 'cat', 'deer',
               'dog', 'frog', 'horse', 'ship', 'truck']
pred_class, conf = predict_image(model, 'test_image.jpg', class_names)
print(f"Predicted: {pred_class}, Confidence: {conf:.4f}")
```
