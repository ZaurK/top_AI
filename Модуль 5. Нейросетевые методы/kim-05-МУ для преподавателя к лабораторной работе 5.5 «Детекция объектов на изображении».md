### 4. Рекомендации по проведению отдельных разделов

#### 4.1. Вступительная лекция

**Ключевые темы:**
1. **Задача детекции:** Что это, зачем нужна.
2. **Bounding boxes:** Форматы (xyxy, xywh, cxcywh).
3. **Якорные прямоугольники (Anchor boxes).**
4. **YOLO:** Принцип работы (разбиение на сетку, предсказание).
5. **Non-Maximum Suppression (NMS):** Удаление дублирующихся框.
6. **Метрики:** IoU, AP, mAP.

**Визуальный материал:** Схема работы YOLO, примеры NMS.

#### 4.2. Использование YOLO

```python
from ultralytics import YOLO

model = YOLO('yolov8n.pt')
results = model('image.jpg', conf=0.5, iou=0.45)
```

**Обсуждение параметров:**
- `conf`: порог уверенности. Слишком низкий — много ложных срабатываний; слишком высокий — пропуск объектов.
- `iou`: порог для NMS. Низкий — меньше дублей; высокий — больше框.

#### 4.3. Параметры детекции

**Эксперимент:** Попросить студентов протестировать разные `conf` и `iou` на одном изображении и визуализировать результаты.

**Ожидаемый вывод:** Студенты должны увидеть, как меняется количество и качество детекций.

#### 4.4. Обучение на кастомном датасете

**Шаги:**
1. Подготовить изображения и аннотации в формате YOLO.
2. Создать файл `data.yaml`:
```yaml
train: dataset/images/train
val: dataset/images/val
nc: 3
names: ['class1', 'class2', 'class3']
```
3. Запустить обучение:
```python
model.train(data='dataset/data.yaml', epochs=50)
```

**Важно:** Для обучения кастомной модели требуется достаточно данных (минимум 100–200 изображений на класс).

### 5. Типичные ошибки

| Ошибка | Решение |
|--------|---------|
| Слишком много ложных срабатываний | Увеличить confidence threshold |
| Пропуск объектов | Уменьшить confidence threshold |
| Дублирующиеся框 | Увеличить iou threshold в NMS |
| Неправильные координаты | Проверить формат аннотаций |
| Модель не видит объекты | Проверить нормализацию изображений |

### 6. Контрольные вопросы

1. **В чём отличие детекции объектов от классификации?**  
   *Ответ:* Детекция — локализация объектов на изображении (bounding box + класс), классификация — только определение класса всего изображения.

2. **Как работает YOLO?**  
   *Ответ:* Разбивает изображение на сетку, каждая ячейка предсказывает bounding boxes и вероятности классов.

3. **Что такое Non-Maximum Suppression?**  
   *Ответ:* Алгоритм удаления дублирующихся框, оставляющий только один с максимальной уверенностью для каждого объекта.

4. **Как вычисляется IoU?**  
   *Ответ:* Отношение площади пересечения к площади объединения двух прямоугольников.

5. **Что такое mAP?**  
   *Ответ:* Среднее значение AP (Area under Precision-Recall curve) по всем классам.
