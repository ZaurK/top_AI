## Система оценки результатов обучения
Результаты обучения рекомендуется оценивать с помощью ФОС по системе РИТМ.
ФОС включает в себя:
1)	лабораторные работы (обязательные виды работ) для текущего контроля;
2)	проектную работу (дополнительная работа);
3)	тесты для семестрового контроля;
4)    экзаменационные тесты.
 
Текущий контроль проводится по итогам защиты выполненных лабораторных работ. Каждая лабораторная работа оценивается 0-100 баллов с порогом 65 баллов. При превышении этого порога баллы накапливаются.
 
Для контроля равномерности работы обучающихся в течение семестра вводятся аттестационные недели – 7-я и 12-я. Для каждой аттестации преподавателем назначается пороговый балл. Пороги с распределением баллов по отдельным видам работ оформляется в виде технологической карты.
 
Для подведения итогов промежуточной аттестации каждый модуль дисциплины оценивается как взвешенная сумма баллов лабораторных работ, нормированная к 100 баллам. Каждый модуль дисциплины считается засчитанным при результате не менее 65%.
Весь текущий контроль оценивается как Nт – взвешенная сумма баллов, нормированная к 60.
 
Кроме обязательных видов работ, обучающиеся могут выполнять дополнительные работы по выбору: проектная деятельность, участие в олимпиадах по ИИ, исследовательская работа с представлением рефератов и выступлением с докладами на конференциях, решение заданий повышенной сложности, выполнение комплексных усложненных лабораторных работ и т.п.
Особенностью дополнительных работ по выбору является то, что за их выполнение обучающийся может получить не более 15 баллов (Nд) по шкале промежуточной аттестации, или ¼ от максимального числа баллов за обязательные виды работ.
 
По итогам работы обучающегося в семестре по системе РИТМ оценка по дисциплине может быть выставлена автоматически, но только после прохождения специальной контрольной акции – семестрового контроля.
Семестровый контроль проводится в форме тестирования, охватывающего все темы дисциплины, и оценивается в диапазоне 10-20 (Ncк).
При своевременной промежуточной аттестации баллы за все виды работ определяются по формуле
Nс=Nт+Ncк+Nд.
Оценка выставляется автоматически по системе РИТМ: 50 -порог тройки, 75 – хорошо, 90 – отлично.
Если студент захочет повысить оценку, то сдает экзамен в виде итогового теста по всем темам с возможностью набрать от 0 до 40 баллов. Если студент набирает не менее 20 баллов, то экзамен может быть засчитан.
Итоговый балл в этом случае рассчитывается следующим образом:
Nс=Nт+Nд+Nэ.
Оценка имеет те же пороги: 50 -порог тройки, 75 – хорошо, 90 – отлично.
Если полученный балл не повышает оценку, то действует оценка по первой формуле.
 
Неуспевающие студенты должны сдавать экзамен.
