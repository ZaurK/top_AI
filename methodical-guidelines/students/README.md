# МЕТОДИЧЕСКИЕ УКАЗАНИЯ ДЛЯ ОБУЧАЮЩИХСЯ ПО ОСВОЕНИЮ УЧЕБНОЙ ДИСЦИПЛИНЫ
## Структура дисциплины
Дисциплина включает обязательные для выполнения части:
Часть 1. Теоретические разделы;
Часть 2. Компьютерный практикум;
Часть 3. Итоговая проверка освоения всех разделов дисциплины.
Дополнительная работа заключается в проектной деятельности, выполнении лабораторных заданий или кейсов повышенной сложности.

## Теоретические разделы
В данной части осваиваются фундаментальные понятия, подходы и методы решения задач компьютерного зрения.  
Теоретические разделы дисциплины изучаются на лекциях и (или) самостоятельно с помощью видеолекций, презентаций, текстового материала и контрольно-тренировочных тестов в одноименном разделе электронного курса (ЭК) на образовательном портале.  
В конце каждого подраздела дается список рекомендованных источников:
Все пройденные элементы каждой темы теоретических разделов автоматически учитываются на ЭК. Уровень освоение отображается в ведомости оценок ЭК (0-20 баллов). Поскольку не исключается учеба по внешним материалам, то баллы за пройденные элементы этой части курса считаются как дополнительные.  

## Компьютерный практикум
Компьютерный практикум посвящен освоению технологий для автоматизации решения задач компьютерного зрения.
Каждая лабораторная работа содержит несколько заданий с методическими указаниями в прикрепленных файлах.  Баллы за лабораторные работы выставляются по результатам защиты.
Выполненные задания в виде файлов документов и кода прикрепляются на электронном курсе с помощью инструмента «Добавить ответ на задание». Все прикрепленные файлы могут послужить в роли портфолио обучающегося для работодателя.
Каждая лабораторная работа оценивается по 100 балльной шкале. Критерии оценивания приводятся в методических указаниях к лабораторным работам. Баллы накапливаются и в конце семестра суммарный бал нормируется к 60 баллам.

## Итоговая проверка освоения всех разделов дисциплины
Итоговая проверка освоения дисциплины проводится в виде семестрового контроля и  / или экзамена в форме компьютерного тестирования по всем модулям.
Семестровый контроль обычно проводится на заключительном аудиторном занятии семестра по дисциплине с оценкой до 20 баллов и порогом в 10 баллов.. К нему допускаются студенты, не имеющие задолженностей по компьютерному практикуму.
При успешном прохождении семестрового контроля оценка выставляется автоматически.
Экзамен проводится согласно расписанию сессии и оценивается автоматически в диапазоне 0..40 баллов. Экзамен сдают студенты, не уложившиеся в график учебного процесса с защитой лабораторных работ, а также студенты, желающие повысить автоматическую оценку после семестрового контроля. На экзамене решаются более сложные варианты тестов. Понижение оценки экзаменом после семестрового контроля не предусмотрено.
В итоговом тесте дается две попытки. Первая попытка может рассматриваться как ознакомительная и выполняться дома самостоятельно. Результирующей считается последняя попытка.

## Общая оценка дисциплины
К семестровому контролю и экзамену должны быть выполнены все лабораторные работы электронного курса, а также допускается одна ознакомительная попытка итогового теста.
Баллы по всем частям дисциплины суммируются.
Зачет по дисциплине выставляется, когда суммарный балл по всем ее частям составляет не менее 60.
Перевод баллов в «оценки» по дисциплине ставятся следующим образом:
удовлетворительно: от 60 до 75 баллов; хорошо: от 75 до 90 баллов; отлично: от 90 балов и выше.

## Взаимодействие при изучении дисциплины
Для взаимодействия при изучении дисциплины на электронном курсе предусмотрены два ресурса:
Объявления – содержат все объявления преподавателей для студентов этого электронного курса. При размещении преподавателем объявления оно автоматически рассылается на электронную почту каждому студенту, которую указал студент при настройке своего профиля на образовательном портале.  
Форум– позволяет любому участнику электронного курса формировать свои новые темы, инициируя обсуждения внутри данных тем. Это позволяет взаимодействовать как с преподавателем, так и студентам между собой.
