### 4. Рекомендации по проведению отдельных разделов

#### 4.1. Загрузка и сохранение (Pillow)

**На что обратить внимание:**
- Различие в форматах (JPEG – сжатие с потерями, PNG – сжатие без потерь, поддержка прозрачности).
- Параметр `quality` при сохранении JPEG – показать, как изменяется размер файла.
- Режимы `RGB`, `L` (градации серого), `RGBA` (с альфа-каналом).

**Типичные ошибки:**
- Попытка сохранить изображение в формате, не поддерживающем альфа-канал (например, JPEG с `RGBA`). – Объясните необходимость преобразования в `RGB`.

#### 4.2. OpenCV и преобразование цветовых пространств

**Ключевой момент:**  
Подчеркните, что `cv2.imread()` возвращает изображение в **BGR**. Это часто вызывает путаницу при визуализации через Matplotlib. Предложите студентам самостоятельно провести эксперимент: вывести на экран изображение в BGR без преобразования и с преобразованием.

**Полезно:**
- Показать, как получить отдельные каналы B, G, R с помощью срезов.
- Сравнить `cv2.cvtColor` с непосредственным изменением порядка срезов (`img[:,:,::-1]`).

#### 4.3. Визуализация Matplotlib

**Рекомендации:**
- Научить отключать оси (`axis('off')`).
- Для grayscale изображений обязательно указывать `cmap='gray'`, иначе Matplotlib применит цветовую карту по умолчанию (viridis), что исказит восприятие.
- Использовать `plt.subplots` для отображения нескольких вариантов (оригинал, инвертированный, каждый канал отдельно).

#### 4.4. NumPy и SciPy

**Основные упражнения, которые стоит сделать совместно:**
- Вычисление средней яркости изображения (через `np.mean()`).
- Изменение контраста путём линейного растяжения гистограммы (использование `np.clip`).
- Применение фильтра Гаусса из `scipy.ndimage` – показать разницу между разными значениями `sigma`.

**Дидактический приём:** Попросите студентов загрузить встроенное изображение `scipy.datasets.face()` и выполнить на нём все операции.

#### 4.5. Рисование на изображениях

**Важные замечания:**
- В OpenCV цвет передаётся как (B,G,R), а не (R,G,B). Дайте студентам цветовую палитру для запоминания: (255,0,0) – синий (OpenCV) / красный (Pillow), (0,255,0) – зелёный, (0,0,255) – красный (OpenCV) / синий (Pillow).
- Для рисования текста в OpenCV можно использовать разные шрифты (`cv2.FONT_HERSHEY_SIMPLEX` и др.) – покажите примеры.
- В Pillow для текста требуется загружать шрифт через `ImageFont.truetype` – при отсутствии системного шрифта используйте дефолтный.

**Задание для творчества:**  
Предложите студентам нанести на изображение аннотацию (например, подпись объекта или рамку вокруг лица). Это повышает мотивацию.

#### 4.6. Работа с видео

**Технические нюансы:**
- В цикле `while cap.isOpened()` обязательно обрабатывать случай, когда `ret == False` (конец видео).
- Для выхода по клавише `q` использовать `cv2.waitKey(1) & 0xFF == ord('q')`. Объясните, почему используется маска `0xFF`.
- Показать, как получить свойства видео (`CAP_PROP_FPS`, `CAP_PROP_FRAME_WIDTH` и т.д.).

**Дополнительное задание (для сильных студентов):**

### 5. Типичные ошибки и способы их предотвращения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Изображение не отображается или отображается с искажёнными цветами | Перепутаны порядки каналов (RGB vs BGR) | Напомнить о необходимости преобразования `cv2.cvtColor(img, cv2.COLOR_BGR2RGB)` перед передачей в Matplotlib |
| Ошибка `TypeError: ...` при рисовании текста | В Pillow не указан шрифт или он не найден | Использовать `ImageFont.load_default()` или проверять наличие шрифта |
| Видео открывается, но кадры не читаются | Неверный путь к файлу или неподдерживаемый кодек | Проверить путь; использовать абсолютный путь; попробовать другой видеофайл |
| Индексация пикселей приводит к ошибке `IndexError` | Использование координат (x, y) вместо (y, x) в NumPy | Напомнить, что в массивах сначала строка (высота), затем столбец (ширина) |
| При сохранении изображения теряется качество | Не указан параметр качества для JPEG | Пояснить, что `quality` по умолчанию 95, но можно регулировать |
| Не отображается окно OpenCV (`cv2.imshow`) | Не вызван `cv2.waitKey()` | Обязательно добавить задержку для обновления окна; после всех окон – `cv2.destroyAllWindows()` |

### 8. Контрольные вопросы для защиты (с ответами для преподавателя)

1. **В чём разница между RGB и BGR?**  
   *Ответ:* Это разные порядки расположения цветовых каналов. OpenCV использует BGR, а большинство других библиотек – RGB. При передаче данных между библиотеками нужно преобразовывать.

2. **Как получить доступ к синей составляющей пикселя в изображении OpenCV?**  
   *Ответ:* `pixel = img[y, x, 0]` (индекс 0 соответствует синему каналу в BGR).

3. **Какие параметры можно указать при сохранении JPEG в Pillow?**  
   *Ответ:* `quality` (от 0 до 100), `optimize=True`, `progressive=True`.

4. **Как в OpenCV рисовать залитый круг?**  
   *Ответ:* В `cv2.circle` передать `thickness=-1`.

5. **Почему в цикле обработки видео используется `waitKey(1)`?**  
   *Ответ:* Это необходимо для обновления окна и обработки событий клавиатуры. Значение 1 – задержка в миллисекундах, достаточная для отображения кадра.

6. **Как преобразовать цветное изображение в чёрно-белое в OpenCV?**  
   *Ответ:* `cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)`.

7. **Какой цветовой картой в Matplotlib следует пользоваться для отображения grayscale?**  
   *Ответ:* `cmap='gray'`.

8. **Что произойдёт, если сохранить изображение с альфа-каналом в формате JPEG?**  
   *Ответ:* Альфа-канал будет отброшен, и изображение сохранится как RGB.
