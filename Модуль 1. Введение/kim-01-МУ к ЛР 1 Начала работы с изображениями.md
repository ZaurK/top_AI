### 2. Теоретическая часть

#### 2.1. Представление изображений в Python

Изображение в Python чаще всего представляется как **многомерный массив NumPy**. Для цветного изображения в формате RGB это трёхмерный массив размера `(height, width, channels)`, где каждый пиксель кодируется тремя значениями (интенсивности красного, зелёного и синего каналов) в диапазоне 0–255.

#### 2.2. Обзор библиотек

| Библиотека | Назначение | Особенности |
|------------|------------|-------------|
| **Pillow (PIL)** | Базовая работа с изображениями | Загрузка/сохранение, изменение размера, простые преобразования. Использует цветовую модель **RGB**. |
| **OpenCV (cv2)** | Компьютерное зрение | Мощный инструмент для обработки изображений и видео. Использует цветовую модель **BGR** (обратный порядок каналов по сравнению с RGB). |
| **Matplotlib** | Визуализация | Отображение изображений, построение графиков. Ожидает данные в формате **RGB**. |
| **NumPy** | Работа с массивами | Базовый инструмент для хранения и манипуляции данными изображений. |
| **SciPy** | Научные вычисления | Модуль `scipy.ndimage` предоставляет функции для обработки многомерных массивов. |

#### 2.3. Цветовые модели: важное замечание

**Критическое различие!** OpenCV хранит изображения в формате **BGR** (Blue-Green-Red), тогда как большинство других библиотек (Pillow, Matplotlib) ожидают **RGB**.

При передаче изображения из OpenCV в Matplotlib для отображения необходимо выполнить преобразование:

```python
image_rgb = cv2.cvtColor(image_bgr, cv2.COLOR_BGR2RGB)
```

Игнорирование этого правила приведёт к некорректному отображению цветов (синий и красный каналы будут перепутаны).

#### 3.6. Задание 5. Рисование на изображениях

**Цель:** освоить инструменты рисования геометрических фигур, стрелок и текста на изображениях с использованием OpenCV и Pillow.

##### 5.1. Рисование с OpenCV

**Важно:** OpenCV использует формат BGR для указания цвета.

```python
import cv2
import numpy as np

# Создание пустого изображения (чёрный фон)
img = np.zeros((500, 500, 3), dtype=np.uint8)

# Линия: cv2.line(изображение, точка_начала, точка_конца, цвет(BGR), толщина)
cv2.line(img, (50, 50), (450, 450), (255, 0, 0), 5)  # синяя линия

# Прямоугольник: cv2.rectangle(изображение, верхний_левый, нижний_правый, цвет, толщина)
cv2.rectangle(img, (100, 100), (400, 400), (0, 255, 0), 3)  # зелёный прямоугольник

# Круг: cv2.circle(изображение, центр, радиус, цвет, толщина)
cv2.circle(img, (250, 250), 100, (0, 0, 255), -1)  # -1 = заливка, красный

# Стрелка: cv2.arrowedLine(изображение, точка_начала, точка_конца, цвет, толщина, tipLength)
cv2.arrowedLine(img, (50, 100), (450, 100), (0, 255, 255), 3, tipLength=0.1)

# Текст: cv2.putText(изображение, текст, позиция, шрифт, масштаб, цвет, толщина)
cv2.putText(img, "Hello, CV!", (150, 450), cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

cv2.imshow("Drawing", img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

##### 5.2. Рисование с Pillow

Pillow использует формат RGB для цвета:

```python
from PIL import Image, ImageDraw, ImageFont

# Загрузка изображения
img = Image.open("example.jpg")
draw = ImageDraw.Draw(img)

# Рисование линий и фигур (цвет в формате RGB)
draw.line([(50, 50), (450, 450)], fill=(255, 0, 0), width=5)  # красная линия
draw.rectangle([(100, 100), (400, 400)], outline=(0, 255, 0), width=3)
draw.ellipse([(200, 200), (300, 300)], outline=(0, 0, 255), width=2)

# Текст (требуется указание шрифта)
try:
    font = ImageFont.truetype("arial.ttf", 24)
except:
    font = ImageFont.load_default()
draw.text((150, 450), "Hello, Pillow!", fill=(255, 255, 255), font=font)

img.save("pillow_drawing.jpg")
```

#### 3.7. Задание 6. Работа с видео (OpenCV)

**Цель:** освоить чтение видеофайлов и видеопотока с камеры, покадровую обработку.

**Ход выполнения:**

1. Чтение видеофайла:

```python
import cv2

cap = cv2.VideoCapture("video.mp4")  # или 0 для веб-камеры

while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        print("Конец видео или ошибка чтения")
        break
    
    # Обработка кадра (например, преобразование в градации серого)
    gray_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    
    # Отображение
    cv2.imshow("Video", gray_frame)
    
    # Выход по нажатию 'q'
    if cv2.waitKey(25) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```

2. Получение информации о видео:

```python
fps = cap.get(cv2.CAP_PROP_FPS)
width = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
height = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
total_frames = int(cap.get(cv2.CAP_PROP_FRAME_COUNT))
print(f"FPS: {fps}, Размер: {width}x{height}, Кадров: {total_frames}")
```
