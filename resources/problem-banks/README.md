## 📚 Университетские курсы с открытыми заданиями

Многие ведущие вузы выкладывают свои материалы в открытый доступ — это готовые структурированные банки заданий с четкими формулировками и критериями оценки.

| Название курса / Университет | Ссылка | Описание заданий |
| :--- | :--- | :--- |
| **COL780 Computer Vision**, IIT Delhi | [github.com/bansal-yash/COL780-Computer-Vision](https://github.com/bansal-yash/COL780-Computer-Vision) | Задания сочетают классические и глубокие методы CV для решения реальных задач обработки и классификации изображений. |
| **CS231n: Deep Learning for CV**, Stanford | [github.com/YYZhang2025/Stanford-CS231N](https://github.com/YYZhang2025/Stanford-CS231N) | Три культовых задания: базовые операции и классификация изображений, сверточные сети и работа с PyTorch. |
| **CSL7360 Computer Vision**, IIT Jodhpur | [github.com/kwanit1142/Computer-Vision-on-different-scenarios](https://github.com/kwanit1142/Computer-Vision-on-different-scenarios) | Лабораторные работы: классические методы обработки, поиск различий на сшитых изображениях, анализ сцен. |

| **CSCI 5722 Computer Vision**, CU Boulder | [github.com/horseisfish/LandingLensClass](https://github.com/horseisfish/LandingLensClass) | Jupyter-туториалы по практическому применению моделей для детекции, сегментации и OCR. |
| **COS 429 Computer Vision**, Princeton | [www.cs.princeton.edu](https://www.cs.princeton.edu) | Расчетные задачи: модель камеры-обскуры, реализация алгоритма Canny с нуля, вычисление градиентов. |

## 💡 Рекомендации по использованию

1. **Для начинающих**: начните с `vision-engineering-exercises` или лабораторных работ из российских практикумов — они дают пошаговые инструкции.

2. **Для углубленного изучения**: возьмите задания из CS231n — это золотой стандарт в области глубокого обучения для CV.

3. **Для подготовки к собеседованиям**: репозиторий `vision-engineering-exercises` создавался специально для портфолио и интервью.

4. **Для проектной деятельности**: используйте кейсы из `computer-vision-case-studies` или задания по детекции дефектов.
